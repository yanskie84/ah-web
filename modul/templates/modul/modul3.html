Website Statis

Website statis adalah website yang isinya statis atau tidak berubah. Situs web statis ini seperti brosur. Perbedaannya adalah brosur dicetak dan didistribusikan sementara situs web statis dikelola dan diakses melalui internet. Sebagai aturan umum, setelah situs web statis aktif di Internet, situs web jarang mengubah kontennya. Seringkali, situs web statis ini hanya terdiri dari HTML dengan halaman hyperlink, tanpa pemrograman sisi server, karena tujuannya terbatas untuk menampilkan informasi di Internet.

HTML (Hyper Text Markup Language)

HTML (Hyper Text Markup Language) merupakan sebuah bahasa markup atau bahasa formatting yang paling dasar dalam pembangunan atau pembuatan sebuah halaman website. Bahasa Markup adalah bahasa komputer yang berfungsi atau digunakan untuk menyusun, memformat, hingga mendefinisikan berbagai teks dan simbol pada dokumen.

Kelebihan dari HTML

  • Mudah dipelajari dan digunakan.
  • Memiliki banyak sumber dengan komunitas yang sangat besar dan penggunaannya luas.
  • Dapat dijalankan di setiap web browser.
  • Dapat dengan mudah diintegrasikan dengan bahasa pemrograman lainnya seperti PHP, Perl, JavaScript, VBScript dan lainnya.
  • Mempunyai Bahasa Markup yang rapi dan konsisten.

Kekurangan dari HTML

  • Tidak dapat berdiri sendiri sebagai bahasa pemrograman web.
  • Tidak memiliki kemampuan interaktif inheren. Interaktif sendiri maksudnya client bisa berinteraksi dengan server. Sehingga untuk mendapatkannya harus menyisipkan bahasa pemrograman yang bisa menangani masalah tersebut.
  • Penggunaan tag yang banyak.
  • Fitur baru tidak dapat dijalankan secara cepat di sebagian browser.

Django

Django adalah framework web Python yang telah mengukuhkan posisinya sebagai salah satu alat yang paling efisien dan lengkap untuk pengembangan aplikasi web. Dengan kumpulan fitur yang luas, Django menyediakan seperangkat alat yang mempercepat proses pembuatan aplikasi web tanpa mengorbankan fleksibilitas dan kontrol. Salah satu keunggulan utama Django adalah ORM (Object Relational Mapping) terpadu yang dimilikinya.
