GUI & PySide2

Pengertian GUI

Graphical User Interface (GUI) atau antarmuka pengguna grafis merupakan suatu bentuk antarmuka pengguna yang memungkinkan pengguna berinteraksi dengan peranti elektronik melalui objek-objek grafis, seperti ikon, penunjuk mouse, tombol, dan menu. GUI lebih memudahkan pengguna dalam melaksanakan suatu tindakan karena cukup mengandalkan mouse dan mengarahkan penunjuk mouse ke objek-objek grafis. Melalui GUI, navigasi ke seluruh desktop dapat dilakukan dengan mudah

Pengertian PySide2

PySide2 adalah modul resmi dari Qt untuk Python Project, yang menyediakan akses ke Qt framework yang lengkap. PySide2 menduplikasi sintaksis dan fungsionalitas PyQt5 dengan sangat sedikit perbedaan. PySide diatur oleh lisensi LGPL yang berarti bahwa kode sumber hanya perlu dirilis ketika kode sumber untuk PySide2 telah diubah

Kelebihan PySide2

  • Pustaka mudah dipelajari meskipun sangat luas.
  • Basis kode tidak terlalu rumit atau panjang.
  • Aplikasi grafisnya lengkap.
  • Lisensi lebih fleksibel.
  • Kompatibel dengan Qt Designer.

Kekurangan PySide2

  • Belum ada integrasi web yang mudah.
  • Komunitas lebih kecil disbanding PyQt.
  • Dokumentasi Python kurang lengkap.
  • Ukuran file output yang cenderung besar.

QtCharts

QtCharts adalah modul yang menyediakan serangkaian komponen bagan yang mudah digunakan. QtCharts digunakan untuk memvisualisasikan data 2D dengan komponen yang mencakup berbagai opsi visualisasi 2D seperti line charts, bar charts, pie charts, scatter charts, area charts, dan banyak lagi. Masing-masing jenis charts ini dilengkapi dengan berbagai opsi penyesuaian, yang memungkinkan untuk melakukan penyesuaian tampilan agar sesuai dengan kebutuhan. Tujuan dari QtCharts adalah untuk menyajikan data dengan cara yang lebih intuitif dan mudah digunakan

Contoh Script

from PySide2.QtWidgets import QApplication, QLabel

app = QApplication([])
label = QLabel("Halo, ini GUI PySide2!")
label.show()
app.exec_()
