Hasil Praktikum

Modul 1:Python Dasar

Pada modul pertama yaitu Python dasar yang membahas tentang Python sebagai bahasa pemrograman yang mudah dipelajari, VSCode sebagai editor teks, flowchart untuk memvisualisasikan alur dari program yang dibuat, dan membuat program sederhana mengenai data perusahaan PT. Indomobil Multi Jasa Tbk.

Modul 2:GUI (Grapichal User Interface)

Pada modul kedua yaitu GUI (Graphical User Interface) yang membahas tentang pembuatan antarmuka grafis menggunakan PySide2 dan QtCharts, serta membuat laman penyewaan PT. Indomobil Multi Jasa Tbk mulai dari login, pemilihan truk, penyewaan, hingga cetak struk dan grafik penyewaan.

Modul 3:Website Statis

Pada modul ketiga yaitu website statis yang membahas tentang pembuatan halaman web menggunakan HTML dan framework Django. Website ini menampilkan informasi perusahaan PT. Indomobil Multi Jasa Tbk yaitu daftar unit truk, fitur pemesanan, dan evaluasi untuk layanan.

Modul 4:Website Dinamis

Pada modul keempat yaitu website dinamis yang membahas tentang penggunaan database, XAMPP, Apache, dan MySQL untuk menyimpan dan mengelola data pemesanan. Website ini menampilkan halaman tabel data, formulir pemesanan, dan profil perusahaan yang memungkinkan untuk interaksi dua arah antara pelanggan dan sistem, sehingga memudahkan dalam pengelolaan data.
